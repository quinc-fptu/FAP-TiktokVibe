# NeoFAP - TikTok Dark Vibe FPT Academic Portal
Reference from: https://github.com/hieuvume/fap-beautifier

Extension làm đẹp giao diện trang FPT Academic Portal (FAP) với phong cách thiết kế TikTok Dark Vibe cực kỳ sang xịn mịn và tối giản.

## 🚀 Tính năng nổi bật
* **Giao diện tối (Dark Mode)** lấy cảm hứng từ TikTok, bảo vệ mắt khi học tập và tra cứu ban đêm.
* **Smart Status Strip** cập nhật trực quan tình trạng điểm danh ngay trên Dashboard.
* **Thời khóa biểu ngày mới (Today's Agenda)** hiển thị chi tiết lớp học và lịch thi trong ngày hôm nay theo múi giờ Việt Nam.
* **Vòng đếm ngược Deadline** tiện lợi, tính toán chính xác thời gian thi cử và nộp đơn từ.
* **Tối ưu hóa dung lượng** loại bỏ hoàn toàn các asset thừa, giúp extension tải cực nhanh (chỉ ~1.8MB).

## 🛠️ Hướng dẫn cài đặt

### Cách 1: Cài đặt từ file đóng gói ZIP (Khuyên dùng)
1. Tải tệp tin [neofap-extension.zip](neofap-extension.zip) từ kho lưu trữ này về máy tính của bạn.
2. Giải nén tệp tin ZIP ra một thư mục (ví dụ đặt tên thư mục là `neofap-extension`).
3. Mở trình duyệt Chrome (hoặc Edge, Brave, Coccoc) và truy cập trang quản lý extension: `chrome://extensions/`.
4. Bật chế độ nhà phát triển (**Developer mode**) ở góc trên cùng bên phải.
5. Nhấp vào nút **Tải tiện ích đã giải nén** (Load unpacked) ở góc trên cùng bên trái.
6. Chọn thư mục `neofap-extension` vừa giải nén ở bước 2.
7. Truy cập `fap.fpt.edu.vn` để tận hưởng giao diện mới!

### Cách 2: Cài đặt trực tiếp qua Git Clone
1. Mở Terminal / CMD và clone kho lưu trữ này về máy:
   ```bash
   git clone https://github.com/quinc-fptu/FAP-TiktokVibe.git
   ```
2. Mở trình duyệt Chrome, truy cập `chrome://extensions/`.
3. Bật chế độ nhà phát triển (**Developer mode**).
4. Nhấp vào nút **Tải tiện ích đã giải nén** (Load unpacked).
5. Chọn thư mục `FAP-TiktokVibe` vừa clone về ở bước 1.
6. Xong!
